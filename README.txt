Enables the Html tags on post titles of the gutenberg editor.

== Description ==

This is a simple plugin that helps user to enable the Html tags on gutenberg editor post titles.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `html-tags-on-post-title` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
